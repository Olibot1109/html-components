# HTML Components

Load HTML components from files and build pages dynamically.

- **[Full Documentation](md/DOCUMENTATION.md)** - Complete API reference and advanced features
- **[Quick Start Guide](md/QUICKSTART.md)** - Get Started

## Installation

Download `html-components.js` or use the CDN:

```html
<script src="https://html-components.vapp.uk/html-components.js"></script>
```

## Development

Run a local server to avoid CORS errors:

```bash
npx http-server -p 8080 --cors
```

## File Structure

```
my-app/
├── html-components.js
├── index.html
├── components/
│   ├── header.html
│   └── footer.html
└── styles/
    └── main.css
```
